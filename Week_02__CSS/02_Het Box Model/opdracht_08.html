Opdracht 0

Welke CSS property kun je gebruiken om ervoor te zorgen dat de waarde van width en height de totale breedte en hoogte is van het blok van de voorgaande opdracht?

"box-sizing" rule declaration

Because if we do not declare this rule the total width and height of the box will be affected, meaning BORDER and PADDING will be calculated.

/* box-sizing: border-box; */
