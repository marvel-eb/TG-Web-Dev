Opdracht 0

Wat zijn de eigenschappen van block boxes en inline boxes?

Block boxes: - Neemt de volledige breedte in.

Block Elemnet Lijst

- p
- h1, h2, h3, h4, h5, h6
- ol, ul
- pre
- address
- blockquote
- dl
- div
- fieldset
- form
- hr
- noscript
- table

Inline boxes: - is block-level element en begin niet op nieuwe regels.

Inline Element Lijst

- b, big, i, small, tt
- abbr, acronym, cite, code, dfn, em, kbd, strong, samp, var
- a, bdo, br, img, map, object, q, script, span, sub, sup
- button, input, label, select, textarea
