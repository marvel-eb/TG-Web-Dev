Opdracht 01

Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

  • inline: - styling direct door in de begintag van het HTML element
  • internal stylesheet: - We voegen STYLES in de head-sectie van het HTML-document
  • external stylesheet: - Styling wordt gelinkt naar een extern bestand / heel handig /
