1. Selectors & Visual rules

Opdracht 03

Er zijn verschillende selectors, bijvoorbeeld;
● type selectors
● descendant selectors
● class selectors

Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

  • type selector: - is een element Selector, zoals "" p, <span>, form en <div> " .het worden over algemeen gebruikt om "brede lijn" wijzigingen

    Example:
    p, span, div { color:#f1f1f1; background-color:#101010; }

  • descendant selectors: - In het volgende voorbeeld worden alle "p" en "h1" -elementen binnen <div> - elementen geselecteerd:

    Example:
    div h1, div p {
    color:#f1f1f1;
    padding:25px;
    }

  • class selectors: - De "".class-selector" selecteert elementen met een specifiek class-attribuut

    Example:
    <h2 > class="heading" >Title </h2 >
    - .heading is CLASS selector
