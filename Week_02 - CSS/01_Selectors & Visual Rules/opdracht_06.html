1. Selectors & Visual rules

Opdracht 06

Verder heb je ook nog de volgende selectors:
- child selectors
- adjacent selectors
- general selectors

Maak in een aparte html bestand voor elke selector een voorbeeld en maak een link in je antwoord naar dat html bestand. Geef in je antwoord ook aan wat de voordelen zijn van elk selector.

Child selector:

- we kunnen een element targeten dat direct onder een element valt.


Example:
ul> li {
background-color:#f1f1f1;
color:#f2f2f2f;
}

Adjacent sibling selector: - selecteert een element dat volgt op een specifiek element.

Example :
div + p {
background-color:#f1f1f1;
color:#f2f2f2f;
}

General sibling selector: - selects all elements that are siblings of a specified element.

Example :
div ~ h1 {
color: darkred;
}
