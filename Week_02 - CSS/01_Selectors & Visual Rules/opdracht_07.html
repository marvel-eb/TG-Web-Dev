Maak nu een html bestand aan met voorbeelden van alle soorten selectors erin. Maak een link in je antwoord die naar dat html bestand verwijst.

element

CSS selectors :

id (#) selector

#heading {
background-color: #8F1594;
}

class selector

.heading {
background-color: #8F1594;
}

universal selector

* {
background-color: #8F1594;
}

descendant selector (space)

div h1 {
background-color: #8F1594;
}

child selector (>)

div > h1 {
background-color: #8F1594;
}

adjacent sibling selector (+)

div + ph1{
background-color: #8F1594;
}

general sibling selector (~)

div ~ h1 {
background-color: #8F1594;
}

Reference: https://www.w3schools.com/css/css_combinators.asp
