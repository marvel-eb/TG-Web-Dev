The Cascade

Cascade by order of importance, If different styles have the same level of specificity and are conflicting, the last rule wins.

Example:
-1. Inline Style
-2. Embeded (internal)
-3. External
-4. User style sheet
-5. Default Browser style

Inheritance

.inheritance {
font-family: helvetica;
fontsize:25px;
color:green;
border:10px dashed red;
}

In this example, the "p" is child of "CLASS - inheritance" meaning it inherits from its parent - but it does not mean all get automatically inherited (only font-family, font-size and color willl be INHERITED not the BORDER.
