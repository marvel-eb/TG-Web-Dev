4. Pseudo elements

Opdracht 01

Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

Pseudo Elements

::after = insert something AFTER the content
::before = insert something BEFORE the content
::first-letter = select the first LETTER of "P"
::first-line = select the first LINE of "P"
::marker = selects MARKER of list items
::selection = select PORTION OF elements
