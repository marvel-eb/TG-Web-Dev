4. Pseudo elements

Opdracht 03

Er zijn aldus 14 verschillende pseudo elementen. Je hebt ze in de voorgaande opdracht reeds opgezocht en uitgezocht wat ze doen. Kun je van al deze pseudo-elementen een voorbeeld maken?

Pseudo Elements

::after
Example: p::before { content: "HI"; color: blue; }

::before
Example: p::before { content: "HI"; color: red; }

::first-letter
Example:p::first-letter { color: #8A2BE2; }

::first-line
Example:p::first-line { color: blue; }

::marker
Example: ::marker { color: blue; font-size: 1.2em; }

::selection
Example: ::selection { background-color: cyan; }

::placeholder
Example: ::placeholder { color: blue; font-size: 1.5em;

} ::cue | ::cue( )
Example: ::cue { color: yellow; font-weight: bold; }

Pseudo Classes

:active
Example : a:active {color:red; }

:hover
Example: a:hover{ color:red; }

:visited
a:visited{ color:red; }

:required
Example: input:required { border: 1px dashed red; }

:link
Example: a:link { color: red; }

:focus
Example: input:focus { color: red; }
