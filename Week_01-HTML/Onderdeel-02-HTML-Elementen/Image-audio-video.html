1. Bij het inladen van een bestand is het noodzakelijk om de locatie van het bestand aan te geven. Dit kan op 2 manieren:
● Absolute path
● Relative path
Wat houden deze 2 paths in? Een image element is voor het inladen van een afbeelding. Afbeeldingen kunnen van verschillende formaat zijn. Bijvoorbeeld, een afbeelding kan een JPG formaat zijn.

- Absolute path : Full URL link address from external source

- Relative Path: local link address
